BloodAid

Android Application on Blood donation along with GPS tracking of Donors and Notifications through Email.

Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. --> Have a basic Knowledge on Android Development.

--> Have a basic Knowledge on PHP and working with Mysql server.

--> Be Confident to deal with errors "Nothing is Impossible everything can be done".

See deployment for notes on how to deploy the project on a live system.

Prerequisites

"Android Studio" Download: https://developer.android.com/studio/index.html

"Wamp Server" Download :http://www.wampserver.com/en/

Installing

INSTALL Both Android Studio and Wamp server.

BE sure both are working Prefectly

Running the project

Once both are installed

Download the code from my page in github/BloodAid or from my

Google Drive : https://drive.google.com/open?id=0B-p1H9iIerKaY2o3Q0VEU3F6d2s

1.Copy android_login_api folder to "C:\wamp64\www.."folder.

2.copy BloodAid folder on to "C:\Users\username\AndroidStudioProjects.."folder.

3.Now open android Studio and open BloodAid project from the above directory.

4.once compile the entire project.

5.once check weather the wamp server is running or not and check the php file from server.

6.open the file "C:\Users\username\AndroidStudioProjects\BloodAID\app\src\main\java\ravis\bloodaid\app\AppConfig.java".

7.edit the ip address to your wamp server address.

8.open phpmyadmin in local server

  1. create a new database android_api.
  2. execute the command present in "C:\wamp64\www\android_login_api\db\database.sql" on sql in phpmyadmin in android_api database.

9.Run the BloodAid project on Emulator.

10.check weather the project working correctly or not.
